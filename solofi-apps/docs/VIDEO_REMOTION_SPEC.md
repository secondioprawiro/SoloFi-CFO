# SoloFi CFO — Remotion Build Spec

Addendum to `docs/VIDEO_PITCH_SCRIPT.md`. That file has narration/timing; this
file has what Remotion actually needs to render: scenes, assets, durations,
visual spec. Read both before scaffolding.

## Project setup

- Separate folder: `video/` at repo root — own `package.json`, does not touch
  the backend build (`src/`, `tsconfig.json`).
- `npx create-video@latest video --template hello-world` (or blank), then
  `registerRoot(RemotionRoot)` per Remotion skill convention.
- Resolution: **1920×1080**, **30fps**. Total: 3:00 = **5400 frames**.

## Visual spec (brand match to `assets/solofi-avatar.png`)

- Background: navy gradient `#0B1220` → `#12203A` (same as avatar).
- Accent: gold `#D4AF37`.
- Text: off-white `#F5F5F5`, bold sans (avatar used Arial Bold — any clean
  sans works, e.g. Inter).
- Logo: `assets/solofi-avatar.png`, used on intro/closing only.

## Required assets (must exist before scaffolding — not generated by Remotion)

1. **`video/public/x402-demo.mp4`** — real screen recording of:
   ```bash
   npx tsx scripts/test-x402-payment.ts
   ```
   Full-screen terminal, readable font size (≥18pt), ~75s. Record for real —
   the script already works end-to-end (402 → sign → 200), don't fake it.
2. **`video/public/webhook-cutaway.mp4`** (optional, ~10s) — one `curl`/chat
   call to `/webhook/okx` and its reply. Skip if time-constrained.
3. **`video/public/architecture-diagram.png`** — export of the README
   Mermaid diagram. Mermaid can't render live inside Remotion easily; export
   once via mermaid-cli (`npx @mermaid-js/mermaid-cli -i README.MD... ` or
   paste the diagram into mermaid.live and export PNG/SVG) at ≥2x resolution.
4. **`video/public/logo.png`** — copy of `assets/solofi-avatar.png`.
5. **Voiceover** — decision needed (see below).

## Voiceover — pick one (open decision, no default chosen yet)

- **Record your own voice** — free, no API cost, matches the free-tier-only
  constraint already set for this project. Recommended default.
- **TTS via ElevenLabs** (Remotion's own voiceover skill uses this) — check
  their free-tier limits before committing; do **not** wire in a paid key,
  same rule as the Gemini fallback work.
- Either way: script text is exactly the "VO:" lines in
  `VIDEO_PITCH_SCRIPT.md` — no rewriting needed, just record/generate as-is.

## Captions

Burn in captions matching each VO line as `<Sequence>` text overlays,
bottom-third, gold text on translucent navy bar — same palette as above.
Timing: sync to each scene's frame range from the table.

## Rendering

```bash
npx remotion render src/index.ts MainComposition out/solofi-cfo-pitch.mp4
```
Local render, no API/token cost — safe to iterate on this step freely once
the composition code is written.
